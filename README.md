# Sistema de Estacionamento

Este projeto é um sistema de estacionamento desenvolvido em TypeScript, projetado para gerenciar a entrada e saída de veículos em um estacionamento. O sistema permite que os usuários cadastrem veículos, visualizem os veículos estacionados e registrem a saída dos mesmos.

## Funcionalidades

- **Cadastro de Veículos:** Permite o registro de veículos com informações como nome do proprietário, placa e horário de entrada.
- **Listagem de Veículos:** Exibe uma tabela com todos os veículos atualmente estacionados, mostrando o nome, a placa e o horário de entrada.
- **Remoção de Veículos:** Permite registrar a saída dos veículos e calcular o tempo de permanência.
- **Persistência de Dados:** Utiliza o LocalStorage para armazenar as informações dos veículos, garantindo que os dados sejam mantidos mesmo após o recarregamento da página.

## Tecnologias Utilizadas

- **TypeScript:** Para a implementação da lógica do sistema.
- **HTML/CSS:** Para a criação da interface do usuário.
- **LocalStorage:** Para a persistência de dados no navegador.
